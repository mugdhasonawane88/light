
Steps to setup the environment

1. We used jSON-Server module to access the rest APIS
Installation details : 
$ npm install -g json-server

2. Unzip attached folder

3. Navigate to unzip folder

3. To run JSON server, open terminal and run below command 
$ json-server --watch db.json --port 3001


4.Open another terminal and install the dependencies
$ npm install

5. To run project
$ npm start
